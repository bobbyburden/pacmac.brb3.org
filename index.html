PacMac

pacman for OS X

A port of the binary package manager pacman from Arch Linux to Apple OS X.

Download the Beta

Version: 1.0-beta

Binary Packages

Don't waste time compiling every piece of software on your computer. Install software in seconds, not minutes.

Open Source

pacman is open source and developed by the Arch Linux team.

Sane storage

All files are stored in /usr/local, and only the latest version of packages are installed.

Fast

pacman is fast. Compiling software is slow. Be fast.

El Capitan Ready

Works great with OS X 10.8+, even 10.11. It even plays nicely with rootless

OS X Specific Repository

PacMac provides a pacman repository developed specifically for OS X computers.
